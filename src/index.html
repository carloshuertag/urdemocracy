urdemocracy

urdemocracy is a software tool to help decision making process in political societies focused on a participatory democracy paradigm using near protocol blockchain.

Participatory democracy axes

Informative axis

Organizational, financial, regulatory, scheduling an contact information.

Deliberative axis

Discussion, debates, forums, propositions

Participatory axis

Consulations, Votations, Survays.

Evaluation extra feature

Validate the actions on participation results.
